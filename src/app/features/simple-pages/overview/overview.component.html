Make music in any tuning system in the world.

Argho Tuning is a free and open source standard for defining and using musical tuning systems (often called microtunings), allowing you to write or perform music using scales tuned differently from the Western 12-tone equal temperament piano.

This site hosts a collection of online tuning tools, free downloadable software, and learning resources for composers, music producers, performers, and musicologists.

For music software and library developers, we also provide a set of open source projects to make integrating Argho Tuning support into your software or virtual instruments as simple as possible.
